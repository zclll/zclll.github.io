  2. json方法

    • json.dumps
      将 Python 对象编码成 JSON 字符串（返回转换后的json字符串）
    • json.dump
      把python对象转换成json对象生成一个fp的文件流，和文件相关。python对象转换为字符串并且写入文件
    • json.loads是用来读取字符串,可以把文件打开，用readline()读取一行，然后json.loads()一行。
    • json.load是用来读取文件的，即，将文件打开然后就可以直接读取。
