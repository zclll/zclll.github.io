Classification 分类

Regression 回归

Clustering 非监督分类

Model Selection 模型选择

sk-learn 工作流程

  1. 实例化要使用的特定机器学习模型
  2. 使模型适合训练数据
  3. 使用模型进行预测
  4. 评估预测的准确性

sk-learn 数据集分割

  1. 留出法
    基本思想是将数据集D（即我们获得的所有样本数据）划分为两个互斥的集合，将其中一个作为训练集S，另一个作为验证集T，即D=SUT，S∩T=Φ。在S上训练出模型后，再用T来评估其测试误差，作为泛化误差的估计值；
